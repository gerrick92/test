# v3.0 UI Navigation Patch

Code-only + image-assets patch. It does not include DB files and does not touch data/input or data/db.

Adds:
- Main landing page with expansion logos
- Expansion pages for Vanilla, TBC, WotLK and All Data
- Hearthstone Home
- Gnome Settings
- Settings page with DB statistics
- Sidebar category navigation
- Pagination polish with 5 page-number window
- Item filters only when Type = Items

Copy files over your current project, then run:

python app.py

No re-import needed.
